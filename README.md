# NovaResume

NovaResume is a modern, customizable resume builder web application.

## Features
- Multi-step progress bar for guided resume creation
- Professional typography and visual hierarchy
- Light and dark mode toggle
- Download resume as PDF
- Resume score evaluation with improvement suggestions
- Resume sharing with unique link
- Responsive and beautiful UI

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/Amur-Tigro-cell/NovaResume.git
   ```
2. Open the `NovaResume/index.html` file in your browser.

## Usage
- Fill out each section of the resume builder.
- Use the progress bar to navigate steps.
- Toggle dark mode for preferred appearance.
- Download your resume as PDF.
- Share your resume using the generated link.

## Technologies Used
- HTML, CSS, JavaScript
- html2canvas, jsPDF (for PDF generation)
